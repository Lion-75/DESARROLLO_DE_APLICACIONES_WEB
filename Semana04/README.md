# Proyecto Semana 4 - Desarrollo de Aplicaciones Web

## Descripción
Proyecto educativo que implementa CSS3 avanzado y Bootstrap 5 para crear una página web moderna y completamente responsiva.

## Características Técnicas
- Bootstrap 5.3.0-alpha1
- CSS3 con animaciones y transiciones
- Diseño responsivo con Grid System
- Componentes Bootstrap: Navbar, Cards, Forms, Collapse
- Media queries personalizadas
- Publicado en GitHub Pages
